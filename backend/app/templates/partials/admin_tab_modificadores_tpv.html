No tienes que predecir todas las variantes del cliente. El camarero podrá escribir peticiones libres en el TPV; System MAC interpretará lo claro, descontará la receta base + deltas seguros y dejará en revisión lo ambiguo. Esta pantalla sirve para revisar excepciones y aprender reglas, no para cargarlo todo a mano.

Flujo real con TPV

  1. TPV registra plato + nota libre del camarero: “sin pan, extra queso, ensalada en vez de patatas”.
  2. Al consumarse la venta, System MAC descuenta la receta base.
  3. Aplica automáticamente lo que entiende con confianza.
  4. Lo que no entiende queda aquí como pendiente para aprender una regla.
